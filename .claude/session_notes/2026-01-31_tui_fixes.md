### Fixed: ANSI Color Codes in Menu Header

**Problem**: Whiptail doesn't render ANSI escape codes, showing raw sequences:
```
meshtasticd ^[[32m●^[[0m  rnsd ^[[32m●^[[0m
```

**Solution**: Added `plain` parameter to `get_status_line()` in `startup_checks.py`:
- `plain=True` returns text indicators: `meshtasticd: UP  rnsd: --`
- `plain=False` (default) returns ANSI colored output for terminal use

**Files changed**:
- `src/launcher_tui/startup_checks.py` - Added `plain` parameter to `get_status_line()`
- `src/launcher_tui/main.py` - Updated `_get_menu_status_hint()` to use `plain=True`

**Result**: Menu now displays cleanly:
```
meshtasticd: UP  rnsd: UP
```
